Bienvenue dans la Simulation de pH-mètre!

Objectif : Cette simulation vous permet de mesurer et de comprendre le pH des solutions en utilisant des outils virtuels comme une burette, une pipette et un pH-mètre.

Étapes pour utiliser la simulation :

  • Sélectionnez une solution dans le menu déroulant.
  • Utilisez la pipette pour ajouter la solution sélectionnée dans la burette.
  • Utilisez le robinet d'eau pour ajouter de l'eau à la solution dans la burette.
  • Utilisez le pH-mètre pour mesurer le pH de la solution.
  • Cliquez sur le robinet de vidange pour vider la burette si nécessaire.
  • Réinitialisez l'expérience en cliquant sur le bouton de réinitialisation.

Utilité :

  • Cette simulation interactive aide à renforcer vos connaissances sur les concepts de pH et de titration.
  • En manipulant virtuellement les outils de laboratoire, vous pouvez observer comment les changements de concentration affectent le pH d'une solution.
