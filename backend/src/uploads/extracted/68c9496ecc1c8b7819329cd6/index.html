Bienvenue dans l'interface de visualisation des acides aminés!

Objectif : Cette interface vous permet de visualiser des modèles 3D d'acides aminés.

  • Sélectionnez un acide aminé dans le menu déroulant.
  • Utilisez la souris pour zoomer et faire tourner la molécule.
  • Référez-vous à la description et à la légende pour plus de détails.
